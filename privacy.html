Privacy Policy

Last updated: January 28, 2026

1. Introduction

FBIM ("we", "our", or "us") provides software solutions for content automation, analytics, and API integrations with social media platforms. This Privacy Policy explains how information is collected, used, and protected when using our services.

2. Information We Collect

  • Authorized Account Data: Information made available through third-party platform APIs when users explicitly authorize access.
  • Usage and Log Data: Technical logs such as timestamps, request metadata, and operational diagnostics.
  • Technical Information: IP address, browser type, and device information used for security and system integrity.

3. How We Use Information

  • Operate and maintain our automation and integration services
  • Execute authorized actions via connected APIs
  • Monitor system performance and prevent abuse
  • Improve service reliability and functionality

4. Data Sharing

FBIM does not sell or rent personal data. Information may be shared only:

  • With platform providers as required to perform authorized API operations
  • When legally required or to comply with regulatory obligations
  • With explicit user consent

5. Data Security

We apply reasonable technical and organizational safeguards, including encrypted communication, restricted access, and monitoring to protect data against unauthorized access or misuse.

6. User Rights

  • Request access to personal data
  • Request correction or deletion of data
  • Withdraw API authorizations at any time via the respective platform
  • Object to certain data processing activities where applicable

7. Data Retention

Data is retained only for as long as necessary to operate the services or comply with legal requirements. Requests for deletion may be submitted at any time.

8. Legal Compliance

FBIM is designed to comply with applicable data protection regulations, including the General Data Protection Regulation (GDPR) and the California Consumer Privacy Act (CCPA), where applicable.
